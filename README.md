# 華興池生態埤塘導覽

這是「運算思維與程式設計」期末網頁作業。網站以華興池生態埤塘為主題，整理地圖、歷史文化、生態調查、水利地景、變遷現狀、保育未來與互動任務。

## 網頁結構

- `index.html`：首頁、照片輪播與各分類入口。
- `about.html`：主題介紹、研究方向與閱讀順序。
- `pond-map.html`：華興池位置、Google 地圖、水文與周邊環境。
- `history-culture.html`：起源故事與周邊生活。
- `ecology.html`：植物、動物與鳥類、昆蟲與水質。
- `landscape-water.html`：給水系統、池塘構造、排水與循環邏輯。
- `changes-current.html`：土地變化、轉型功能與面臨挑戰。
- `conservation-future.html`：環境教育、生態保育方式與未來規劃。
- `interactive-tasks.html`：選擇題、填充題與分類小遊戲。
- `plants.html`：岸邊植物名錄卡片頁。
- `mimosa.html`：含羞草觀察筆記。
- `clover.html`：酢漿草觀察筆記。
- `plantain.html`：車前草觀察筆記。
- `field-gallery.html`：田野照片牆補充。
- `navigation-demo.html`、`side-layout-demo.html`：保留早期版面練習紀錄，不是主要繳交入口。
